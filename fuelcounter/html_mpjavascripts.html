FUEL COUNTER

Scenario: You and Your friends go on a voyage with Your car. Hardly were You able to agree one specific weekend when You all have some free time and now You can go! You don't want to end up like a bunch of math students however, waving around with sheets of paper and pencils on a parking lot in the middle of nowhere, arguing how much each of You should pay for fuel.
How is this useful: Basing upon provided data the program will count how much each member of the expedition should share for fuel.
Explanation: Prices of petrol change rapidly. You might want to use this program before the trip, as well as after, just to compare...
Source code: This is a second program I wrote in Java Script. Nothing fancy actually, but quite useful I think. I feel there are to many restrictions as far as user's input data is concerned, however this is something I'll improve in the upcoming programs hopefully. Download this app from my GitHub repository or just hit npm i fuelcounter in Your terminal.
